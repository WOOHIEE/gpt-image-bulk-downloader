# Privacy Policy

GPT Image Bulk Downloader does not collect, sell, transmit, or store user data on any external server.

The extension reads the active ChatGPT page only when the user clicks the scan buttons. It uses that page content to find image URLs and nearby prompt text. On `chatgpt.com/images/`, it may request ChatGPT image-library and conversation endpoints from the user's own browser session to locate the user's images and recover prompts. Download processing happens locally in the browser, and prompt metadata is embedded into files before Chrome saves them.

The extension uses Chrome storage only for local settings such as filename templates, folder names, and download options.

No analytics, remote logging, tracking pixels, or third-party scripts are included.

The use of information is limited to the extension's stated purpose: finding ChatGPT images, pairing them with prompt text, and saving local files requested by the user. No prompt, image, browsing, or account data is transmitted to the developer or any external service by this extension.

The use of information received from Chrome extension APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.
